Data acquisition with the Bobcat¶

Department of Mechanical and Materials Engineering Wright State University

Equipment and Software Setup Procedures¶

The following provides instructions for the collection of acceleration data from the cantilever beams in the vibration lab (RC 010) upon excitation from an impulse hammer using Bobcat data acquisition tools.

Channels Configuration¶

  1. Turn on computer
  2. Power on Bobcat modem (switch on back)
  3. Launch Bobcat software. The Local and Viewer windows should appear.
  4. Define channel parameters. On the Local window menu, select:
    1. → Setup
      1. → Channels. The channel definition window should appear. In this window, input the accelerometer serial number and voltage sensitivity for channel 2. These values can be found on a card inside the accelerometer storage case. Also, set the type for channels 3 and 4 to “inactive” as well as the ICP to “on” for channels 1 and 2 as shown in figure 1. Afterwards, select OK.

Figure 1: Channels Configuration Window

  1. Connect the hammer and accelerometer to the Bobcat modem on channels 1 and 2, respectively.
  2. Using wax, affix the accelerometer to the beam at the location where a response will be measured. Tape the wire about an inch behind the sensor to the beam.

Set Response Viewing Parameters¶

  1. Set the plotting parameters for the hammer and accelerometer responses. On the Viewer window menu, select:
    1. → View
      1. Deselect view ports excluding the following:
        1. Plot toolbar
        2. Plot definition
        3. Page Settings
    2. Locate and click the icon shown in figure 2. The set number of rows and columns window should appear. In this window, change the following parameters as indicated. Afterwards, select OK.

Figure 2: Viewer Plots Icon

  1. Rows ›→ 2
  2. Columns ›→ 1
  1. Select the top plot and click the plot reduction icon shown in figure 3 until only “A” remains above the plotting region. Right Click on “A” and change the plot color to black. With the top plot still active, set the following parameters as indicated.

Figure 3: Viewer Plots Icon

  1. Under “Channel” tab ›→ Ch1
  2. Under “Ref Channel” tab ›→ Off
  3. Drop down menu to the upper right of the “Ref Channel” tab ›→ time trace
  4. Drop down menu to the lower right of the “Ref Channel” tab ›→ volts
  1. Select the bottom plot and set the following parameters as indicated.
    1. Under “Channel” tab ›→ Ch2
    2. Under “Ref Channel” tab ›→ Off
    3. Drop down menu to the right of the “Ref Channel” tab ›→ time trace

Equipment Test¶

  1. Set acquisition parameters for equipment test. On the Local window menu, select:
    1. → Setup
      1. → Acquisition. The acquisition window should appear. In this window, change the following acquisition values as shown in figure 4. Afterwards, select OK.
        1. Type ›→ Analysis
        2. Resolution ›→ 1600
        3. Bandwidth ›→ 500
        4. Window Type ›→ None
        5. Num Avgs ›→ 1

Figure 4: Acquisition Window

  1. Set control parameters for data storage. On the Local window menu, select:
    1. → Setup
      1. → Controls. The controls window should appear. In this window, change the following parameters as indicated. Afterwards, select OK.
        1. Data Type ›→ Puma
        2. File Name ›→ Query
  2. Open and dock Analyzer control. On the Local window menu, select:
    1. → View
      1. → Analyzer control. The analyzer control window should appear. Grab this window and dock it to the Local window.
  3. Test the hammer and accelerometer. On the Analyzer control, select:
    1. → Trigger tab. On this tab, change the following parameters as indicated. Afterwards, select Apply.
      1. Source ›→ Free Run
    2. → Control tab. On this tab, take the following actions as instructed to test the hammer.
      1. Be prepared to strike the table with the hammer.
      2. Click the “Start” button.
      3. Gently strike the table a few times within the hammer in the same manor intended to excite the beam for data collection.
      4. Inspect the hammer response (top) plot in the viewer window. Spikes should occur at the time points which the table was struck. Document the amplitude at these voltage peaks. If spikes do not appear, consult the TA.
    3. → Control tab. On this tab, take the following actions as instructed to test the accelerom- eter.
      1. Be prepared to tap the beam with your finger.
      2. Click the “Start” button.
      3. Gently tap the beam a few times with your finger.
      4. Inspect the accelerometer response (bottom) plot in the viewer window. oscillations should occur at the time points which the beam was tapped. If oscillations do not appear, consult the TA.

Data Collection¶

  1. Set acquisition parameters for data collection. On the Local window menu, select:
    1. → Setup
      1. → Acquisition. A pop up window should appear. In this window, change the following parameters as indicated. Afterwards, select OK.
        1. Num Avgs ›→ 5
  2. Set analyzer control parameters. On the Analyzer control, select:
    1. → Trigger tab. On this tab, change the following parameters as indicated. Afterwards, select Apply.
      1. Source ›→ Ch1
      2. Theshold ›→ amount determined in equipment test
      3. Slope ›→ rising
      4. Arming Mode ›→ Manual
      5. Manual Accept ›→ Checked
    2. → Control tab. On this tab, perform the following parameters tasks as indicated to collect data for each case required.
      1. Select “Open Data File” at the bottom of the Analyzer control window. Create a file with a meaningful name (eg: “Case 1”) and save it to the desktop
      2. Click the “Start” button and repeat the following instructions for all five impulses to generate the average response for the current case.
        1. Click the “Manual Arm” button
        2. Gently strike the beam with the hammer.
        3. Inspect the hammer response plot. A single impulse peak should appear. Multiple peaks indicate double hits.
        4. Click “accept” if the hammer response exhibits a single peak, otherwise, click reject and return to step A.
        5. Use your hand to dampen residual vibrations from previous hits between each measurement.
      3. Once all measurements for a case have been conducted, click “Stop”.
      4. Click “Store”.
      5. Select “Close Data File” at the bottom of the Analyzer control window.

Convert Data¶

  1. Once all data has been collect, each file must be converted to a Matlab loadable file format. On windows start menu, select:
    1. → Puma Export Manager. For each data file, perform the following tasks.
      1. Locate data file path in the left window.
      2. Set the output file type as MAT (SDD → MAT).
      3. Identify the output path in the right window.
      4. Execute the file conversion by clicking the arrow between input
